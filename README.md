# FURURE_UX_02 – CarePlus Clinic Appointment Booking UI

## About the Project
This project is a mobile appointment booking UI designed for CarePlus Clinic using Figma as part of the Future Interns UI/UX task.

## Problem
Many local clinics still manage appointments using:

- Phone calls
- WhatsApp messages
- Manual registers

This leads to:

- Missed bookings
- Scheduling confusion
- Poor customer experience
- Lost revenue

## Solution
This mobile UI helps patients easily book appointments and helps clinics manage time efficiently.

Features:

- Splash Screen
- Login Screen
- Doctor Selection
- Date and Time Booking
- Appointment Confirmation
- Contact Screen

## Target Users

- Clinic patients
- First-time visitors
- Local customers
- People who want easy appointment booking
